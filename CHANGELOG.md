# Changelog

All notable changes to MetaSvelte will be documented in this file.

## [1.0.0] - 2024-01-15

### Added
- Initial release
- Core `SeoHead` component
- Global `seoStore` for state management
- Type-safe SEO configuration
- Helper functions:
  - `createSeoConfig()`
  - `createArticleSeo()`
  - `createProductSeo()`
  - `createOrganizationSeo()`
  - `mergeSeoConfigs()`
- Support untuk:
  - Base meta tags
  - Open Graph (Facebook, LinkedIn, WhatsApp)
  - Twitter Cards
  - Facebook-specific tags
  - JSON-LD structured data
- SEO presets untuk use cases umum
- Full TypeScript support
- Svelte 5 runes support
- Comprehensive documentation
